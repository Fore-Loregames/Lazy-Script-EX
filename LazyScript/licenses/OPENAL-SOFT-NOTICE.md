# OpenAL Soft runtime notice

The OpenAL Soft runtime is not embedded in this archive. `setup-runtime.bat` downloads the official OpenAL Soft binary distribution and copies its 64-bit DLL into `runtime/OpenAL32.dll`. The setup script also copies the license file from that distribution when present.
