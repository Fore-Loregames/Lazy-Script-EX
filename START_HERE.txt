LazyScriptEX 0.21.6

1. Run setup-runtime.bat
2. Run INSTALL_VSCODE_EXTENSION.bat
3. Run open-api.bat
4. Create a project with new-project.bat MyGame
5. Build Projects\MyGame\build.bat

OpenGL is the default graphics backend. Call LSG.use_vulkan() before LSG.open() to use Vulkan with the same LSG/LSSL front-end API.
